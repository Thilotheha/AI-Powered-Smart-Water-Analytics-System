# 🚰 AI-Powered Smart Water Analytics System

An intelligent water purification analytics platform built using Python, Streamlit, Power BI, and IoT simulation concepts.

## 📌 Project Overview

This project simulates a smart water purification monitoring system capable of:

- Monitoring water quality parameters
- Predicting operational risk
- Estimating membrane lifespan
- Recommending suitable purifiers
- Simulating IoT-based live sensor monitoring
- Visualizing insights using Power BI dashboards

---

## 🚀 Features

### ✅ Streamlit Application
- Live IoT sensor simulation
- Real-time TDS monitoring
- Water hardness analysis
- Turbidity monitoring
- Temperature tracking
- Operational risk prediction
- Membrane lifespan estimation
- Purifier recommendation engine

### ✅ Power BI Dashboard
- Operational risk distribution
- Water quality analytics
- Purifier recommendation insights
- Interactive slicers
- KPI tracking
- Business intelligence visualization
